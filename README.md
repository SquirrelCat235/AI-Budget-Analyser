# AI-Budget-Analyser
An AI-powered budget analyzer that predicts future expenses and provides personalized spending recommendations based on historical data.

## How to run locally

If you'd like to test this on your own machine, follow these steps:

### Prerequisites
Make sure you have [Python 3.8+](https://www.python.org/downloads/) installed.

### 1. Clone the repository
Open a terminal and run:
```bash
git clone https://github.com/SquirrelCat235/AI-Budget-Analyser.git
cd AI-Budget-Analyser
```

### 2. Set up a virtual environment (Recommended)
**Windows:**
```cmd
python -m venv venv
venv\Scripts\activate
```

**Mac/Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Install requirements
```bash
pip install -r requirements.txt
```

### 4. Run the application
```bash
streamlit run app.py
```

The application will start, and your default web browser will automatically open a local tab showing the dashboard (typically `http://localhost:8501`).
